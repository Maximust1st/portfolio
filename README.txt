Maxwell Ajala Multi-Page Portfolio

How to update:
1. Edit index.html for the homepage.
2. Edit projects.html for the project list.
3. Edit files inside /projects/ for each project case study.
4. Put screenshots inside /assets/images/.
5. Put videos inside /assets/videos/.
6. Put your resume PDF inside /assets/files/ and name it resume.pdf.

To deploy:
Upload all files and folders to your GitHub portfolio repo.
Vercel will redeploy automatically.
