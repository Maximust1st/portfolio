Overview

Project Summary

An Android-based digital forensics project designed to reconstruct investigative timelines from multiple digital evidence sources.

Problem

What problem does it solve?

Digital investigations often involve scattered evidence from logs, messages, GPS data, metadata, and device activity. Without a clear timeline, investigators can miss relationships between events.

Solution

What I built / learned

This project focuses on collecting and organizing events into a unified timeline, normalizing timestamps, supporting filtering, identifying conflicts, and helping generate a clearer investigative narrative.
