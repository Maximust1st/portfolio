Overview

Project Summary

A cloud security lab focused on understanding S3 bucket exposure, misconfiguration risks, and secure storage practices.

Problem

What problem does it solve?

Misconfigured cloud storage can expose sensitive files, create privacy risks, and give attackers access to business data.

Solution

What I built / learned

This lab documents the process of understanding S3 bucket permissions, identifying risky configurations, and explaining safer cloud storage practices.
